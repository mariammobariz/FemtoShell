# FemtoShell
## run steps
- Go to FemtoShell folder
- Open the terminal
- to view the source file write the following command :
`cat FemtoShell.c`
- to build the program write the following command :
`gcc -o FemtoShell FemtoShell.c `
- to run the program write the following command :
 `./FemtoShell `
